개인정보처리방침

ToolNest 개인정보처리방침

ToolNest는 회원가입 없이 사용할 수 있는 정적 웹 도구 사이트이며, 사용자가 선택한 영상 파일과 입력값을 자체 서버로 전송하거나 저장하지 않습니다.

수집하는 개인정보

현재 ToolNest는 회원가입, 댓글, 결제, 서버 저장 기능을 제공하지 않습니다. 따라서 이름, 이메일, 전화번호 같은 개인정보를 필수로 수집하지 않습니다.

파일과 입력값 처리

타임코드, 영상 길이, 목표 용량, 자막 파일 내용, 선택한 영상 파일은 사용자의 브라우저 안에서 처리됩니다. 영상 클립 편집기에서 선택한 파일은 사용자의 기기에서만 열리며, 결과 파일도 사용자가 직접 다운로드합니다.

로컬 저장소 사용

현재 페이지는 도구 사용에 필요한 기본 상태를 브라우저 안에서만 처리합니다. 사용자가 브라우저 데이터를 삭제하면 사이트 이용 중 생성된 임시 상태도 함께 사라질 수 있습니다.

문의 처리

사용자가 문의 페이지를 통해 직접 제공한 정보가 있는 경우, 해당 정보는 문의 확인과 답변 목적에 한해 사용됩니다. 불필요한 개인정보는 문의 내용에 포함하지 않는 것을 권장합니다.

방침 변경

사이트 기능이나 운영 방식이 변경되면 개인정보처리방침도 함께 업데이트될 수 있습니다. 변경된 내용은 이 페이지에 게시됩니다.
